Step 1: Spring Start the container
(ApplicationContext context = new AnnotationConfigApplicationContext(AppConfig.class)) iska matlab hum bol rahe hai ioc container ko chaalu karo using the annotaion based configuratiions 

Step 2: Spring reads AppConfig.java
(phir AppConfig.class ko read karta hai jismein rules likhe hai ki aapko kaun kaun se object banane hai and kya karna hai )

Step 3: Spring processes @ComponentScan.
(iski wajah se wo saare )

Step 4: Spring finds @Component classes.
(matlab ab wo in do class ko search karega )

Solution1:@Primary notation:ye kya hai ki aap jsiko bhi chahte ho ki main mera iska banna chahiye jab bhi ioc container confuse ho then aap use karo @Primary jismein ko bhi aapko most priority dena hai 

jaise aap upi payment ko jayada priority dena chhate ho uss class ke paas jaake likho do @Primary 


@Component
@Primary
public class UpiPayment implements PaymentService{
    @Override
    public void pay()
    {
        System.out.println("upi payment sucessful");
    }
}

Solution2:using @Qualifier annotaion :ye kya hai ki sabse pahle aap kya karo ki aapke paas jo bhi clasess hai uspe jaake likho @Qualifier dono pe 
and jisko jeetwana hai ya jisko bhi priorityDena hai aap usko de skate ho and uske baad jo aapka jo bhi class hai jo aapko inject karna chah raha hai usske constructor mein bhi aap likh do @Qualifier
