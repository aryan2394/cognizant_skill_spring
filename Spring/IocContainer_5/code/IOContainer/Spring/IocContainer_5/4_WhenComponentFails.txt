Matlab ki ioc container can craete the objects(beans) in 2 ways 

1:@Component:these is class annotaion matlab ki jis class pe hum normally normally ioc container ko bol sakte hai ki aap beans craete and manage sab karo 

2:@Bean:these is method annotaion matlab ye bhi beans(object) hi banata hai lekin ye uska object khud hi banata hai then paas  to ioc container to handle it 
iska hum use karte hai jab kisi class pe hum @Component nahi laga sakte hai
