Life Cycle

1. Clean Lifecycle (clean)

2. Default Lifecycle
validate → compile → test → package → verify → install → deploy

3. Site Lifecycle (site)

Clean Lifecycle → Cleans/deletes previous build files.
Default Lifecycle → Builds your project from validation to deployment.
Site Lifecycle → Generates project documentation (project reports, JavaDocs, etc.).
