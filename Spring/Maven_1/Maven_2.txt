Interview Definition (Simple)

Maven is a Build Automation Tool and Dependency Management Tool for Java projects.

It:

📁 Maintains a standard project structure.
📦 Downloads required libraries (dependencies).
⚙️ Compiles Java code.
🧪 Runs tests.
📦 Packages the application into a JAR/WAR file.
