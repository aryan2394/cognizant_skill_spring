option1:@Value:jis bhi varibale ko aap chahte ho ki wo value application.properties se lele waha ye likh dena 

Agar tum khud property lena chaho

Suppose

student.name=Aryan(these is in application.properties likha hai toh )

Java me

@Value("${student.name}")
private String name;


OPTION2:ye @Value wala bada wo hai heactic maanlo 100 jagah hai values leni hai toh aap @Value karke tohda nahi likhoge har varibale ke paas 

Ab @ConfigurationProperties

Ye aur powerful feature hai.

Suppose

application.properties
student.name=Aryan
student.age=22
student.city=Delhi

Aur class

@Component
@ConfigurationProperties(prefix = "student")
public class Student {

    private String name;
    private int age;
    private String city;

    // getters setters
}
