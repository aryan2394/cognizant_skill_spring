First, what is a Filter?

A Filter is a component that executes before the request reaches the Controller and can also execute after the Controller sends the response.

Think of it as a security guard at the entrance of your application.

Client
   │
   ▼
Filter
   │
   ▼
Dispatcher Servlet
   │
   ▼
Controller
   │
   ▼
Service
   │
   ▼
Repository

A filter can:

Check authentication
Log requests
Add headers
Validate requests
Modify requests/responses
Block requests

How to create the filter and kaise pata chalega ki ye Controller se bhi pahle chalega

@Component
public class LoggingFilter implements Filter {

    @Override
    public void doFilter(ServletRequest servletRequest,
                         ServletResponse servletResponse,
                         FilterChain filterChain)
            throws IOException, ServletException {

        System.out.println("Request entered in Logging filter");

        filterChain.doFilter(servletRequest, servletResponse);
    }
}
