SABSE IMPORTANT AND POWERFUL Configuration:

@EnableAutoConfiguration: Spring Boot ka "smart assistant" hai. Ye project ki dependencies aur environment ko dekhkar automatically required configuration classes load karta hai aur unke beans IoC Container mein register kar deta hai, taaki tumhe manual configuration na likhni pade.

Flow yaad rakhne ke liye:
SpringApplication.run()
          │
          ▼
IoC Container start
          │
          ▼
@EnableAutoConfiguration
          │
          ▼
Classpath (dependencies) check
          │
          ▼
Matching AutoConfiguration classes load
          │
          ▼
@Bean methods execute
          │
          ▼
Beans IoC Container mein register
          │
          ▼
