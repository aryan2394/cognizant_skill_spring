1:static methods :ko overide karenge toh kya hoga:

sabse pahle understaning the static matlab isko class se matlab hai object se nahi 

class Car{
    static void fnc()
    {
        System.out.println("class");
    }
}

class ElectricCar extends Car{
    static void fnc()
    {
        System.out.println("subclass");
    }
}

class Main{
    public static void main(String[] args)
    {
        Car c=new ElectricCar();
        c.fnc();
        // output:class
        //why because static methods are bind to the class not to object 
        // matlab ab jitne class ko inehrit karke overide kardo lekin jo fnc hai wo class function hai usse object se matalab nahi hai wo hamesha answer according to the class dega 

    }
}
