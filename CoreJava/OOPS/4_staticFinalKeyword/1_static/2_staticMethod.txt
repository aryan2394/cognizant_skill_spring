Static Method ki Sabse Important Rules
Rule 1
Static Method Static Variable ko Direct Access Kar Sakta Hai.
class Student {

    static String college = "IIT Guwahati";

    static void printCollege() {

        System.out.println(college);

    }

}

✔ Correct

Kyuki

Dono class ke members hain.

Rule 2
Static Method Direct Instance Variable Access Nahi Kar Sakta.
class Student {

    String name;

    static void printName() {

        System.out.println(name);

    }

}

❌ Error

Kyun?

Rule 3
Static Method Direct Non-static Method ko Call Nahi Kar Sakta.
class Student {

    void display() {

    }

    static void print() {

        display();

    }

}

❌ Error

Kyun?

Non-static method kisi object ka hota hai.

Static method ke paas koi object hi nahi.

Agar Call Karna Ho
class Student {

    void display() {

    }

    static void print() {

        Student s = new Student();

        s.display();

    }

}

✔ Correct

Static Method ka Real-Life Example

Sabse famous static method

public static void main(String args[])

Ye static kyon hai?

Jab program start hota hai,

abhi tak

Demo obj = new Demo();

banaya hi nahi gaya.

Toh JVM ko koi method chahiye jo

object ke bina hi chal sake.

Isi liye

main()

ko static banaya gaya.

JVM directly call karti hai:

Demo.main(args);

Object banane ki zarurat nahi padti.

Static Method Kab Banate Hain?

Jab method:

Kisi object ke data par depend na kare.
Sirf class-level ka kaam kare.
Utility function ho.
Common operation perform kare.

Examples:

Math.max(10, 20);
Math.sqrt(25);
Integer.parseInt("123");
Arrays.sort(arr);

Ye sab static methods hain, kyunki inhe chalane ke liye object ki zarurat nahi hoti.

⭐ Golden Rule (Interview Trick)

Static Method belongs to the Class, not to any Object.

Isliye:

Object ke bina call ho sakta hai.
Direct sirf static members ko access kar sakta hai.
Non-static members ko access karne ke liye pehle object banana padta hai.
this keyword use nahi kar sakta, kyunki static method ke paas current object hota hi nahi.
