Accessing Static Variable

Wrong (although Java allows it)

s1.college = "IIT Guwahati";

Preferred

Student.college = "IIT Guwahati";

Kyun?

Kyuki

College class ka variable hai.

Object ka nahi.
Example
class Student {

    String name;
    int age;
    int rollNo;

    static String college = "IIT Guwahati";
}

public class Demo {

    public static void main(String args[]) {

        Student s1 = new Student();
        Student s2 = new Student();

        s1.name = "Aryan";
        s2.name = "Rahul";

        System.out.println(Student.college);

    }

}

Output

IIT Guwahati
Agar College Change Kare
Student.college = "IIT Delhi";
