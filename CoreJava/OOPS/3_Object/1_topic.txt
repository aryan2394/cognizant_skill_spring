Ek object exactly kitna space lega?

👉 Java mein iska ek fixed answer nahi hota.

Kyun?

Kyuki object ka total size =

Object Size =
Object Header
+ Instance Variables
+ Padding (Alignment)

Chaliye detail mein samajhte hain.

Example
class Student {
    String name;
    int age;
    int rollNo;
    String college;
}
Step 1: Instance Variables ki Size

Assume 64-bit JVM.

Variable	Size
String name	8 bytes (reference)
int age	4 bytes
int rollNo	4 bytes
String college	8 bytes

Total fields:

8 + 4 + 4 + 8 = 24 bytes

Lekin object sirf 24 bytes ka nahi hota.

Step 2: Object Header

Har object ke saath JVM kuch extra information bhi rakhti hai.

Jaise:

Object lock information
Hashcode
Kis class ka object hai

Ye sab Object Header mein hota hai.

Modern 64-bit JVM mein generally

Object Header ≈ 12–16 bytes

(Exact JVM implementation par depend karta hai.)

Step 3: Padding

JVM memory ko align karti hai.

Suppose

Header = 12 bytes

Fields = 24 bytes

Total = 36 bytes

Lekin JVM object ko 8-byte boundary par align karti hai.

Toh

36

ko

40 bytes

ya

48 bytes

tak round kiya ja sakta hai.

Isko Padding kehte hain.

Final
