Exception

Definition:
A recoverable (handleable) abnormal condition
that occurs during program execution.(matlab code execute karte samay wo galti huwi hai and hum usko sahi kar sakte hai jaise ki /0 toh hum inout lete samay hi bol denge 0 is not allowed ya aap ye galti sahi kar lo)

Examples:
1. File is missing
2. Wrong input
3. Divide by zero

Example:
int num = a / b;

errors:iska matlab hai ki aab aapke code mein error aa gaya hai matlab ab aap isko sahi nahi kar sakte hai jaise ki error mein StackOverflowError(jab stack fill ho jaata hai) ya OutOfMemoryError(ye tab aata hai jab heap bhar jaata hai)

How to handle? it 

sabse pahle samajte hai flow of exception hota kaise hai ?and gar hum usko handle nahi karenge toh kaun karta hai usko handle 

Exception hamesha us method se start hoti hai jahan error hua. Agar us method me handle nahi hoti, to call stack ke through parent method → uske parent → ... → main() tak propagate karti hai.
 Agar main() me bhi handle na ho, to JVM exception ka poora stack trace print karke program terminate kar deti hai.

public class Demo3 {

    public static void main(String[] args) {

        // Exception handling

        System.out.println("Step 1");

        try {
            int a = 5;
            int b = 0;

            System.out.println(a / b);   // Exception yahin aayegi
        }
        catch (ArithmeticException e) {
            System.out.println("Division by zero is not allowed");
        }

        System.out.println("Step 2");
    }
}
